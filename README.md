This project is developed as part of **Task 3** of the internship.  
It implements **search and pagination** using **PHP and MySQL** with a simple user interface.

## Technologies Used
- PHP  
- MySQL  
- HTML  
- CSS  

## Features
- Search posts by title or content  
- Pagination for posts  
- Improved UI using CSS  

## Project Files
- `index.php` – Main file  
- `config.php` – Database connection  
- `style.css` – Styling  

## How to Run
1. Start Apache and MySQL in XAMPP  
2. Create database `blog_db`  
3. Place project folder in `htdocs`  
4. Open browser:http://localhost/task3-php-blog/index.php
